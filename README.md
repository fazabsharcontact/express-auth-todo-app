# Express Auth Todo App

A simple todo web application built with **Node.js**, **Express.js**, and **EJS** featuring session-based authentication and protected routes.

This project was created to practice backend fundamentals such as authentication, middleware, sessions, and CRUD operations without using a database.

---

## Features

- User registration & login
- Session-based authentication
- Protected routes using middleware
- Create & delete todos
- Todos scoped per user
- Data persistence using JSON files
- Reusable layout with EJS partials
- Simple and clean UI

---

## Tech Stack

- Node.js
- Express.js
- EJS
- express-session
- File System (JSON storage)
- Tailwind CSS (CDN)
